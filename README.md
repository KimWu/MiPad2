# 小米平板2 刷 MIUI 終極教程

### 如想知道最新的刷機情報, 請關注本人的微博:  http://www.weibo.com/KingHeroWu

## **注意事項**
1. 刷機會清除所有資料； 
2. 平板最少有30%電量.
3. USB最好插電腦後面的插口， 儘量不要插前面的.
4. 解開的包，儘量用比較短的路徑，例如: c:\MiPad2, 不要有空格、中文和符號。 
5. 最好做完所有windows 更新並重新電腦, 並保證安裝了主機板的驅動. 
6. 不要使用精簡版的Windows XP , windows 7, 會導致安裝不了驅動. 
7. MiFlash的驅動並不適合小米平板2, 如已經安裝, 請刪除並重啟. 
8. 無論之前是windows, MIUI,還是其他Linux系統， 都可以直接刷為MIUI.
